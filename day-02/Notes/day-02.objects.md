 ### What is it?

A non-primitive data types that stores data, things or information in  keys and value pairs basically.

### Any cool stuff to know about it? 

So basically those "things" I mentioned earlier are called _Properties_

And what are these **Properties** ?

They made up of "keys:value" pair. The key -> is a string while the value -> can be any data type.
